LabVIEW Web Services

The LabVIEW development environment and any applications built from VIs contain an integrated Web server. Besides publishing the same type of documents as other Web servers, the built-in Web server also can publish pictures of running VIs.

You can configure the LabVIEW Web Server manually in the Options dialog box. You also can configure the Web server using the Application property node. Refer to the LabVIEW Help for more information about configuring the LabVIEW Web Server.

Web Server: Configuration

By default, the Web server is disabled. To turn on the Web server, select Tools»Options to display the Options dialog box. Select Web Server: Configuration Options from the top pull-down menu and place a checkmark in the Enable Web Server checkbox. Using the default settings, you now can connect to the server from a browser by entering http://host.domain.com into the browser's URL field (replace host.domain.com with the internet address of your computer, or, if the server and the browser are on the same computer, use the address localhost.

Note: By default, the Unix version of LabVIEW is configured to use Port 8000. This allows the LabVIEW Web Server to run in accounts that do not have root access. If you have root access, you can change the HTTP Port to 80. If you use any port other than 80, you must specify the port number in URLs that refer to your server.

Web Server: Browser Access

By default, all computers have access to your Web server. To specify which computers on the Internet can access your Web server, select Tools»Options to display the Options dialog box. Select Web Server: Browser Access from the top pull-down menu. You might want to limit access to only a few specific computers (with entries like host1.domain.com and host2.domain.com) or to all computers within a domain (with an entry like *.domain.com).

Web Server: Visible VIs

By default, all VIs can be viewed. To configure which VIs can be viewed using the Web server, select Tools»Options to display the Options dialog box. Select Web Server: Visible VIs from the top-pull down menu.
